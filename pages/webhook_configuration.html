معلومات مهمة

  • يتم حفظ الإعدادات بشكل آمن في المتصفح (localStorage)
  • تأكد من صحة عناوين Webhook URL قبل الحفظ
  • مفاتيح API سرية - لا تشاركها مع أحد
  • استخدم زر "اختبار الاتصال" للتحقق من صحة الإعدادات
  • يمكنك تفعيل أو تعطيل كل خدمة بشكل منفصل
  • إذا واجهت مشكلة "401 Unauthorized" مع Plate Recognizer، استخدم رابط إعادة تعيين الرمز للحصول على رمز جديد
